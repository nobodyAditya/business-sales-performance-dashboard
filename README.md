# Business Sales Performance Dashboard

## Overview
This project analyzes Superstore sales data using Microsoft Excel and Power BI to identify business trends, profitability drivers, regional performance, and product-level insights.

## Project Objectives
- Analyze sales and profit performance
- Identify top-performing regions and categories
- Evaluate monthly sales trends
- Discover best-selling products and sub-categories
- Generate actionable business recommendations

## Tools Used
- Microsoft Excel
- Power BI
- Pivot Tables
- Data Visualization

## Key Results
- Total Sales: $2.30 Million
- Total Profit: $286.40 Thousand
- Profit Margin: 12.03%
- West Region contributed the highest sales
- Technology was the highest-performing category
- Phones and Chairs were the leading sub-categories

## Files Included
- Business_Sales_Performance_Dashboard.pbix
- Business_Sales_Performance_Report.pdf
- superstore-Analysis.xlsx

## Business Recommendations
- Increase investment in high-performing regions
- Expand Technology product offerings
- Improve planning for peak sales periods
- Focus marketing efforts on top-performing products
